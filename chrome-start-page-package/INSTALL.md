# Chrome Start Page Plugin - 安装指南

## 📦 安装方法

### 步骤1: 准备扩展文件
确保您有以下文件：
- `manifest.json`
- `new-tab.html`
- `popup.html`
- `js/` 文件夹（包含所有JavaScript文件）
- `css/` 文件夹（包含样式文件）
- `icons/` 文件夹（包含图标文件）

### 步骤2: 在Chrome中加载扩展
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 在右上角开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含所有文件的文件夹
6. 扩展将自动安装并启用

### 步骤3: 验证安装
1. 打开新标签页，确认显示自定义首页
2. 浏览器工具栏应该显示扩展图标

## 🧪 功能测试
安装完成后，请按照以下步骤测试核心功能：

### 测试1: 创建分组
1. 点击"添加分组"按钮
2. 输入分组名称（如"工作"）
3. 确认分组出现在网格中

### 测试2: 添加书签
1. 点击"添加书签"按钮
2. 输入标题和网址
3. 选择所属分组
4. 确认书签出现在相应分组中

### 测试3: 数据持久化
1. 创建一些分组和书签
2. 关闭浏览器
3. 重新打开浏览器
4. 打开新标签页
5. 确认数据仍然存在

## 🆘 常见问题
如果遇到问题，请检查：
1. Chrome版本是否为88+
2. 开发者模式是否已启用
3. 文件结构是否完整
4. 控制台是否有错误信息
