# Chrome Start Page - 安装测试指南

## 🧪 快速测试

### 1. 安装扩展
```bash
# 1. 打开扩展管理页面
chrome://extensions/

# 2. 开启"开发者模式" (右上角开关)

# 3. 点击"加载已解压的扩展程序"
# 选择 chrome-extension 文件夹
```

### 2. 验证安装
- [ ] 新标签页显示自定义首页（不是Chrome默认页面）
- [ ] 浏览器工具栏显示扩展图标
- [ ] 页面有"添加分组"和"添加书签"按钮

### 3. 功能测试

#### 分组管理
- [ ] 点击"添加分组" → 输入名称 → 确认分组出现
- [ ] 点击编辑按钮 → 修改名称 → 确认更新
- [ ] 点击删除按钮 → 确认删除

#### 书签操作
- [ ] 点击"添加书签" → 输入标题和网址 → 选择分组
- [ ] 点击书签标题 → 确认能打开网页
- [ ] 点击删除按钮 → 确认书签删除

#### 数据持久化
- [ ] 创建分组和书签
- [ ] 关闭浏览器后重新打开
- [ ] 确认数据仍然存在

#### 邮件书签后台打开
- [ ] 添加邮件书签：`mailto:test@example.com`
- [ ] Cmd+点击 → 邮件在后台窗口打开
- [ ] 右键邮件书签 → 选择"在后台新窗口打开邮件"

## 🔧 常见问题

### 扩展无法加载
**解决**: 检查manifest.json是否存在，确保文件路径不包含中文

### 新标签页没变化
**解决**: 检查扩展是否启用，点击"刷新"按钮，重启浏览器

### 样式显示异常
**解决**: 检查CSS文件路径，清理浏览器缓存

### 数据无法保存
**解决**: 检查manifest.json是否包含storage权限
