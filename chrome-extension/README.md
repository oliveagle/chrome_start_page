# Chrome Start Page Plugin

一个功能强大的Chrome浏览器扩展，替换默认新标签页为自定义首页，支持书签分组管理、背景图片个性化等功能。

## 🌟 特性

### 核心功能 (P1 MVP)
- ✅ **新标签页替换** - 完全替换Chrome默认新标签页
- ✅ **书签分组管理** - 创建多个分组来组织书签
- ✅ **书签CRUD操作** - 添加、编辑、删除、导航书签
- ✅ **数据持久化** - 使用Chrome Storage API本地存储
- ✅ **响应式设计** - 适配不同屏幕尺寸

### 高级功能 (P2/P3)
- 🔄 **图标自动获取** - 自动获取并显示网站favicon
- 🎨 **背景图片自定义** - 个性化设置首页背景
- 🔄 **数据导入导出** - 备份和恢复书签数据
- 🌐 **Pixabay集成** - 获取免费背景图片
- ☁️ **Google账户同步** - 多设备数据同步

## 🚀 快速开始

### 安装步骤

1. **下载扩展文件**
   ```bash
   # 克隆或下载项目到本地
   # 确保包含完整的chrome-extension文件夹
   ```

2. **启用开发者模式**
   - 打开Chrome浏览器
   - 地址栏输入 `chrome://extensions/`
   - 在右上角开启"开发者模式"

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `chrome-extension` 文件夹
   - 扩展将自动安装并启用

4. **验证安装**
   - 打开新标签页，确认显示自定义首页
   - 浏览器工具栏应该显示扩展图标

### 首次使用

1. **打开扩展设置**
   - 点击浏览器工具栏中的扩展图标
   - 或在新标签页点击"设置"按钮

2. **创建分组**
   - 点击"添加分组"按钮
   - 输入分组名称（如"工作"、"学习"等）
   - 可以创建多个不同类别的分组

3. **添加书签**
   - 点击"添加书签"按钮
   - 或在分组中点击"+ 添加书签"
   - 输入网址、标题并选择所属分组

## 📝 使用指南

### 分组管理
- **创建分组**: 点击"添加分组"，输入名称
- **编辑分组**: 点击分组标题旁的编辑按钮
- **删除分组**: 点击删除按钮，确认删除（会同时删除组内所有书签）

### 书签操作
- **添加书签**: 点击"添加书签"或组内的"+ 添加书签"
- **编辑书签**: 点击书签旁的编辑按钮
- **删除书签**: 点击书签旁的删除按钮
- **打开书签**: 直接点击书签标题

### 数据管理
- **导出数据**: 右键点击扩展图标 → "导出数据"
- **导入数据**: 右键点击扩展图标 → "导入数据"

## 🔧 自定义配置

### 修改默认设置
编辑 `js/storage.js` 中的 `getDefaultSettings()` 方法：

```javascript
getDefaultSettings() {
    return {
        theme: 'default',           // 主题颜色
        background: 'gradient',     // 背景类型
        showEmptyGroups: true,      // 显示空分组
        iconSize: '16px',          // 图标大小
        layout: 'grid'             // 布局类型
    };
}
```

### 自定义样式
编辑 `css/styles.css` 来修改：
- 颜色主题
- 字体大小
- 布局间距
- 动画效果

## 🐛 常见问题

### Q: 扩展安装后新标签页没有变化？
A: 检查扩展是否启用，尝试刷新扩展或重启浏览器。

### Q: 书签图标不显示？
A: 网站可能没有favicon，或网络连接问题。手动刷新图标功能将在后续版本中添加。

### Q: 数据丢失怎么办？
A: 定期使用导出功能备份数据。扩展使用Chrome Storage API，数据与浏览器绑定。

### Q: 如何卸载扩展？
A: 在 `chrome://extensions/` 中找到扩展，点击"移除"。
