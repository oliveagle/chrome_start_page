# Chrome Start Page Plugin

一个功能强大的Chrome浏览器扩展，替换默认新标签页为自定义首页，支持书签分组管理。

## 🌟 核心功能

- ✅ **新标签页替换** - 完全替换Chrome默认新标签页
- ✅ **书签分组管理** - 创建多个分组来组织书签
- ✅ **书签CRUD操作** - 添加、编辑、删除、导航书签
- ✅ **数据持久化** - 使用Chrome Storage API本地存储
- ✅ **响应式设计** - 适配不同屏幕尺寸
- ✅ **邮件书签后台打开** - CMD+点击或右键菜单

## 🚀 快速开始

### 安装步骤

1. **启用开发者模式**
   - 打开Chrome，输入 `chrome://extensions/`
   - 开启右上角"开发者模式"

2. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择 `chrome-extension` 文件夹

3. **验证安装**
   - 打开新标签页，显示自定义首页即成功

### 首次使用

1. **创建分组**：点击"添加分组"，输入名称（如"工作"、"学习"）
2. **添加书签**：点击"添加书签"，输入网址、标题并选择分组
3. **打开书签**：直接点击书签标题即可打开

## 📝 使用指南

### 分组管理
- **创建**: 点击"添加分组"按钮
- **编辑**: 点击分组标题旁的编辑按钮
- **删除**: 点击删除按钮，确认删除

### 书签操作
- **添加**: 点击"添加书签"或组内的"+"
- **编辑**: 点击书签旁的编辑按钮
- **删除**: 点击书签旁的删除按钮
- **打开**: 直接点击书签标题

### 邮件书签后台打开
- **CMD/Ctrl + 点击**：邮件链接在后台窗口打开，普通链接在后台标签页打开
- **右键菜单**：右键邮件书签，选择"在后台新窗口打开邮件"

## 🐛 常见问题

### Q: 扩展安装后新标签页没有变化？
A: 检查扩展是否启用，尝试刷新扩展或重启浏览器。

### Q: 数据丢失怎么办？
A: 扩展使用Chrome Storage API，数据与浏览器绑定。建议定期使用导出功能备份。

### Q: 如何卸载扩展？
A: 在 `chrome://extensions/` 中找到扩展，点击"移除"。
