# 邮件书签后台打开功能

## 核心功能

### 1. CMD/Ctrl + 点击
- **邮件链接** → 后台窗口打开
- **普通链接** → 后台标签页打开
- **优势**：无需右键，操作快速

### 2. 右键上下文菜单
- **功能**：右键书签 → "在后台新窗口打开邮件"
- **适用**：邮件链接在后台窗口打开

## 邮件链接识别

扩展自动识别以下邮件链接：
- **协议**：`mailto:user@example.com`
- **域名**：gmail.com, outlook.com, qq.com, 163.com 等
- **关键词**：URL包含 `mail` 或 `email`

## 使用方法

### CMD/Ctrl + 点击
1. 正常点击：当前标签页打开
2. 按住 Cmd/Ctrl 点击：后台打开
   - 邮件链接 → 后台窗口
   - 普通链接 → 后台标签页

### 右键菜单
1. 右键点击邮件书签
2. 选择 "在后台新窗口打开邮件"
3. 邮件在后台新窗口打开

## 修改的文件
- `manifest.json` - 添加 contextMenus 权限
- `js/background.js` - 后台窗口和上下文菜单
- `js/main.js` - CMD/Ctrl 检测和事件处理
- `js/bookmark.js` - 后台打开参数支持
- `new-tab.html` - 添加帮助提示
- `css/styles.css` - 帮助提示样式
